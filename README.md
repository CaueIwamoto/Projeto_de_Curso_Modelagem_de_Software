# Projeto 2 - Empresa de Tecnologia 

## Sobre o Projeto 2 📓:
Será desenvolvido e criado uma sequência de diagramas para este projeto, sendo eles: diagrama de casos de uso, de classes, de sequência, de estado e de atividades. Sendo o conteúdo uma empresa de tecnologia que deseja implementar um sistema de gestão para compartilhamento de micromobilidade urbana (bicicletas e patinetes elétricos). Sendo o objetivo oferecer uma plataforma onde usuários possam localizar veículos disponíveis em um mapa, realizar desbloqueio via aplicativo, utilizar o transporte de forma autônoma e efetuar o pagamento pelo tempo de uso.
